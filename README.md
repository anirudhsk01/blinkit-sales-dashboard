# Blinkit Sales Analysis Dashboard

<img src="Blinkit_Dashboard.png" width="900">

## Project Overview
This project presents an interactive Power BI dashboard analyzing Blinkit grocery sales data.  
The dashboard provides insights into sales performance, outlet distribution, product categories, and customer ratings to support data-driven business decisions.

## Business Requirement
The objective of this project is to analyze Blinkit’s sales performance and identify key trends related to outlet establishment, product categories, and customer ratings using Power BI visualizations.

## KPI Requirements
The following key performance indicators were analyzed:

- **Total Sales** – Overall revenue generated from items sold
- **Average Sales** – Average revenue per sale
- **Number of Items** – Total count of different items sold
- **Average Rating** – Average customer rating for items

## Dashboard Features
The dashboard includes the following analytical views:

- Sales trend by outlet establishment year
- Sales distribution by outlet size
- Sales comparison by outlet location
- Product category sales analysis
- Fat content analysis of products
- Interactive filter panel for outlet location, outlet size, and item type

## Key Insights
- **Total Sales:** $1.20M
- **Average Sales:** $141
- **Number of Items:** 8523
- **Average Rating:** 3.9
